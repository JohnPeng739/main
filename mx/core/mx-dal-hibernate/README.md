## 安装和引用
*Maven*
```xml
<dependency>
    <groupId>org.mx</groupId>
    <artifactId>mx-dal-hibernate</artifactId>
    <version>1.3.8</version>
</dependency>
```
*Gradle*
```gradle
compile 'org.mx:mx-dal-hibernate:1.3.8'
```
